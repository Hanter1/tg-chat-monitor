# Android — TG Chat Monitor

Приложение запускает тот же Python-монитор (Telethon + SQLite) на телефоне
через [Chaquopy](https://chaquo.com/chaquopy/) и держит процесс живым
с помощью **Foreground Service**.

**App-first:** чаты, ключевые слова, мониторинг и журнал — в UI приложения.
Уведомления о совпадениях приходят **из приложения**. Telegram-бот опционален
(дублирование в ЛС при желании).

## Скачать

1. Откройте [Releases](https://github.com/Hanter1/tg-chat-monitor/releases).
2. Скачайте файл `tg-chat-monitor-*-android.apk` (или `*-android-debug.apk`).
3. На телефоне: **Настройки → безопасность → установка неизвестных приложений**.
4. Откройте APK и установите.

## Первый запуск

1. Откройте **TG Chat Monitor**.
2. Нажмите **«Получить API ID и Hash»** — откроется [my.telegram.org/apps](https://my.telegram.org/apps).
3. Войдите, при необходимости создайте приложение, скопируйте `App api_id` и `App api_hash`.
4. Вернитесь в приложение → **«Вставить из буфера»** (или введите вручную) → **Продолжить**.
5. **Старт** → войдите в Telegram (телефон → код → 2FA при необходимости).
6. Вкладки **Чаты** / **Слова** — добавьте группы и ключевые слова.
7. На **Главной** включите мониторинг.

> API ID/Hash Telegram не отдаёт через Bot API — их можно только взять на my.telegram.org.
> Кнопки в приложении открывают страницу и помогают вставить скопированные значения.

### Опционально: бот и дублирование в Telegram

Вкладка **Ещё**:

- `BOT_TOKEN` от [@BotFather](https://t.me/BotFather) (есть кнопка открыть)
- переключатель **Дублировать в Telegram**

Без токена приложение работает полностью самостоятельно.

## Батарея и автозапуск (важно)

Android может убить фоновый процесс. Обязательно:

1. В приложении: **Исключить из оптимизации батареи**.
2. На Xiaomi / HyperOS, Huawei, Oppo, Samsung: разрешите **автозапуск** и снимите
   ограничения расхода батареи для TG Chat Monitor.

## Ограничения

- Телефон должен быть **онлайн**; при выключении или без сети мониторинга нет.
- APK большой (внутри Python runtime).
- Это **не** сервер 24/7 — для постоянной работы надёжнее ПК / VPS / Docker.
- Telethon использует **ваш** аккаунт: вы должны состоять в отслеживаемых чатах.

## Сборка локально

Нужны JDK 17, Android SDK, Python **3.13** (как `buildPython` для Chaquopy):

```bash
cd android
./gradlew :app:assembleDebug
# APK: app/build/outputs/apk/debug/app-debug.apk
```

Для локальной сборки нужны Android-wheels в `android/wheels/`
(`pyaes` уже в репозитории; `pydantic-core` собирается в CI —
см. [releasing.md](releasing.md)).

Release-подпись — см. [releasing.md](releasing.md).
