Настройки подключения

Заполните параметры — они сохранятся в файл .env

Обязательные
Токен бота от @BotFather → команда /newbot
Получить на my.telegram.org/apps
API Hash с my.telegram.org/apps (для Telethon)
Ваш Telegram ID — узнать у @userinfobot
Дополнительные (можно оставить по умолчанию)
Имя файла сессии Telethon (создаётся автоматически при первом запуске)
Путь к SQLite-базе данных
Интервал опроса чатов в секундах (минимум 5)

Установка зависимостей

Установим пакеты из requirements.txt

Команда, которая будет выполнена:
pip install -r requirements.txt

Готово к запуску!

Осталось запустить бота и настроить мониторинг через Telegram

  • 1
    Запустите приложение
    start.bat
    Или ярлык tg-chat-monitor на рабочем столе (создаётся при install.bat). Не вводите python main.py вручную — нужно запускать из папки проекта.
  • 2
    Найдите бота в Telegram и отправьте /start
  • 3
    Добавьте чат — перешлите сообщение из группы и отправьте /add_chat
  • 4
    Добавьте ключевые слова — например: /add_word bitcoin
  • 5
    Запустите мониторинг — /start_monitor
